Proceso de elaboración

El proceso de elaboración de cerveza artesanal comienza con la mezcla de malta de cebada con agua caliente para extraer sus azúcares, lo que llamamos maceración. Luego, este líquido dulce (mosto) se hierve con lúpulo, que le da ese toque amargo y aroma tan especial. Una vez enfriado, se añade la levadura, que fermenta el azúcar en alcohol y CO2. Tras unos días de maduración, ¡ya tienes cerveza lista para disfrutar!

Maceración

La maceración es el primer paso en la elaboración de la cerveza. Consiste en mezclar la malta de cebada con agua caliente para extraer los azúcares fermentables. Este proceso dura aproximadamente una hora y se realiza a una temperatura controlada.

Cocción

Después de la maceración, el mosto se hierve y se le añade lúpulo. La cocción dura entre 60 y 90 minutos y es crucial para esterilizar el mosto y extraer los compuestos amargos y aromáticos del lúpulo.

Fermentación

Una vez enfriado el mosto, se transfiere a un fermentador y se le añade levadura. La fermentación puede durar desde unos pocos días hasta varias semanas, dependiendo del tipo de cerveza que se esté elaborando.

Maduración

Después de la fermentación, la cerveza se deja madurar para que los sabores se desarrollen y se suavicen. Este proceso puede durar desde unas pocas semanas hasta varios meses.

Envasado

Finalmente, la cerveza se envasa en botellas, latas o barriles y está lista para ser disfrutada.
